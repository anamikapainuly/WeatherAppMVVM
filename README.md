# WeatherAppMVVM
Weather App using MVVM
MVVM With Retrofit and Recyclerview in Kotlin
Coroutine Flow, Emit functions , change in data
Room Database
Hilt for DI, database instance, dao instance
Repo, VM
Filter and Query to sort data
